🎒 수학여행 미션 (Firebase)
🔥 Firebase 설정 가이드

5분이면 완료됩니다!

아래 단계를 따라하시면 학생 답변이 실시간으로 Firebase에 저장됩니다.

1Firebase 프로젝트 만들기

  1. Firebase Console 접속
  2. "프로젝트 추가" 클릭
  3. 프로젝트 이름: "수학여행미션" 입력
  4. Google 애널리틱스: 사용 안함 선택 (선택사항)
  5. "프로젝트 만들기" 클릭

2Realtime Database 만들기

  1. 좌측 메뉴에서 "빌드" → "Realtime Database" 클릭
  2. "데이터베이스 만들기" 클릭
  3. 위치: "asia-southeast1" 선택
  4. 보안 규칙: "테스트 모드에서 시작" 선택
  5. "사용 설정" 클릭

3웹 앱 추가 및 설정 복사

  1. 프로젝트 개요 옆 ⚙️ 아이콘 → "프로젝트 설정" 클릭
  2. 아래로 스크롤하여 "내 앱" 섹션 찾기
  3. 웹 앱 아이콘 (</>) 클릭
  4. 앱 닉네임: "미션트래커" 입력
  5. "앱 등록" 클릭
  6. firebaseConfig 부분을 아래 입력란에 붙여넣기

Firebase Console에서 복사한 설정을 그대로 붙여넣으세요

⚠️ 중요: Firebase 설정을 입력해야 학생 답변이 저장됩니다!
